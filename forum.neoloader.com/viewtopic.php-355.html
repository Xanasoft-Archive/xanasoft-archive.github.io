CrashDump NeoLoader v0.53a (Build 167) vom 13.11.2015 (languagefile german, Captcha Dienst 9kw.eu)

Vorgehensweise:
1. Start "NeoLoader"
2. Menü (links) => Services => Captcha Solvers => 9kw.eu
3. Rechtsklick auf "9kw.eu" => Füge Account hinzu
4. 9kw.eu API Key als Benutzername und Passwort hinterlegt. => OK
5. Rechtsklick auf den neuen Account (api key) mit dem angeblichen Status "Ok" => Prüfe Account
6. CRASH! (crashdump siehe Anhang)

Dürfte bei anderen Captcha Diensten ähnlich sein wo ein API Key statt Username+Passwort genutzt wird. Eventuell auch ein allgemeines Problem mit der Funktion um die Accounts zu überprüfen.
