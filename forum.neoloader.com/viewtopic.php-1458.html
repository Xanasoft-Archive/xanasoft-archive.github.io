Posted: Sun May 25, 2014 7:09 am
by DavidXanatos
you select it from the networks combo box,
and optionaly you select the desired download type (all, torrent, ed2k, neo, hosters)

and you enter e search term and press search like with normal search modes.

automated searchmode, currently searches on all networks (depanding on download type)
that means neokad, mulekad, ed2k servers, and online search providers currently only TPB (header Search;Type: btih must be set telling the auto search what kind of downloads can be found using this site)
i will add more providers soon...


For the future i would also like to be able to auto categorize the found files for example by series season, etc...
to make the search more content oriented than file oriented.
for example that it would create a grupe ubuntu, with sub grupes for each version and than only in each sub grupe show the individual files...
but that may be difficult to be done algorithmically...
