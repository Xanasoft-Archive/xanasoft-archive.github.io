With a premium Multihoster account you can download from the supported hosters list, with all the advantages of the traditional premium account. The biggest benefit being that you only need to pay once instead of paying every single supported host.


Apps that can successfully connect to multihosters:
1. jDownloader : Multihoster FAQ - JD Community (Very good, lots of devs)
2. MiPony : Mipony - Features and advantages (Very good, less RAM req)
3. Vity's FRD : FreeRapid Downloader :: FreeRapid Downloader - Plugins (lots of plugins, very active devs)
4. JetDownloader (ok)

From some of the programs above; you can look at the plugin source code if you need help connecting to the multihosters.
And some multihosters actually rewards you, if your software can connect to their service >> Zevera Downloader Developers Information

Make it happen David. We know you're the king of features (your Neomule mods are awesome!).
Besides, I'm sure most people would choose a multihoster (if they know about it) instead of several single hosters.

To start, integrate Zevera (download the API from the link above), test the plugin, make the UI newbie friendly, let Zevera know and they will give you the rewards << Worth around 200-300GBP.


Request API from another multihosters (especially the one that support torrents ex. Real-Debrid, Alldebrid, Putdrive) and everybody benefit.
Pros:
1. For Multihosters: Obviously get more customers. And possibly save some bandwidth during peak hours.
2. For Torrents: Faster completed seeds, so it's easier for people to keep their ratio (on private tracker). And user most likely will keep the file shared; especially if they know it's on an anonymous network.
3. For ed2k: More completed chunks (lower ed2k queue for other requested file faster). And can help with current incomplete shares (if the hash matches).
4. For Neo: Jumpstarting the Neo Anon Network << Very important.
5. For users: Faster download from multiple sources. Save resources, instead of opening several programs at once.
6. More popularity for your software and attract more devs, to help with bugs, features, etc.
