You could test it like this, start one neo with a file or a multifile of which you create more than one torrent in neo, keep in mind that you need to end up with different info hashes else the 2nd torrent creation wil fail, so for example choose different piece sizes for the crations.

Now wait until neo publishes the data into the kad and download each of the torrents with a different torrent client (or with the same should also work).

And than start a new torrent add in it the magnet of oen of the two torrents and neo should find the other torrent and start downloading from booth.
