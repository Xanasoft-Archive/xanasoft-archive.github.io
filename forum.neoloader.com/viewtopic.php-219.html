If neoloader adds magnet link to download file,the problem can't happen.
If neoloader adds neo link to download file,the problem can happen.

I know, the reason is that torrent has a much smaller pieces Effectively than NeoHash,
If your master hash is a torrent hash it will complete the files much faster.

The NeoHash can have a part size down to 256 kb, but due to the tree structure it is not required to operate at the lowest level it can be operated at 256kb * 2^n
And n is selected such that the entire hash set is only around 100kb in size such that it can be stored and retrieved in NeoKad,
in future I will add an option to retrive the higher resolved branches on demand but thats not planned soon.
