hmm thats a problem, the start paused applyes only to the multifile that was added, the sub files are independent and set to started, as otherwise starting the multifile would leave the sub files paused.

I'm not sure how to solve this conflict properly, a sub file can be part of more than one multi file so i can not just replicate the multi file state onto its subfiles.
