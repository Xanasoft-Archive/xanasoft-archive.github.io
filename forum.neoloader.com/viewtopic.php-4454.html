[CENTER][color="#FF0000"]List of all bugs found in NeoLoader v0.25b[/color][/CENTER]

GUI
- Small lags when switch from a window to another (example: Downloads to Shared, Archive to Shared...)
- When you complete a download Details/Transfer/Torrent info remain in the window
- Show double filename in grabber when you add a torrent and after a multi torrent. Is possible that happen also in other place and in other cases.
- Software name is "NeoLoader", in tray icon is "Neo Loader", in forum post is "NeoLaoder". Confused...
- Sometimes upload speed in shared files when upload is 0 remain stucked into an upload value, that is false
- Graph bug, legend say that both lines, red and green, are for torrent upload. Really ed2k line is the red one
- Moving bar to the right, progress bar in transfer list disappear

Torrent
- Seed/Peer balance should be changed because avaibility is always too low [uTorrent do it ? How?]
- Torrent sharing is not working if you move the file downloaded away from Incoming you lose info
- Some torrents don't start without tiers/trackers [Would be useful add DHT and other infos per torrent like in uTorrent]
- Torrent pause don't work they still download datas
- How to change file directory from Incoming, into a shared folder, without lose torrent info for upload?
- If you put a file ABCD.efg in Incoming, and grab again his torrent, NeoLoader start download again and at the end idk what happen...

eD2K
Personally I had no bugs!

Neo
- Neo ask for 1 update that not exist at every start up
--> Update not working because problem with server
- When delete/remove/complete download, folder remain in Temp directory
- A lot of times is not possible connect the NeoKad

Player
- Not working. Got also a crash when testing, .dmp sent to David Xanatos.

OCH
Not tested yet
