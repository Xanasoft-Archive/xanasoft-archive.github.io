Postby DavidXanatos » Tue Jul 02, 2013 10:51 pm

GonZo wrote:Maybe related to the cpu P3 ?

No,
no CPU specific releases are planned.

The advantage of such optimizations are negligible for P2P applications, most stuff neo does is a job for ALU's and most CPU optimizations work with the FPU or vectoring.
For raw Data processing like for example video rendering/compression it helps a lot, but for normal program logic code it does not help.

Where it helps is for example file hashing but afaik the used library is already quite highly optimized in this regard.
