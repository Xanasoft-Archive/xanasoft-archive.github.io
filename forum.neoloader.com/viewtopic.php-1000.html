Actually its not possible setup different speeds or balance with different % between networks.

NeoLoader splits the upload almost randomly, since we doesn't have a "real" queque system. A client on the waiting list its picked up randomly, so in theory the file with more request have more chances to get more upload slots.
There's not a different system for each network except for BT that should (I don't remember exactly, if I'm wrong xanatos will say it in next post) respect TFT while downloading, so while downloading a torrent probably u'll upload only downloading torrent if they are requested from the client that are sending you datas.

Shortly, if you have a more popular bt file than an ed2k one, u'll see that there'll be more upload speed on torrent protocol and reverse.
