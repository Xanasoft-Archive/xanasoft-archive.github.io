Postby DavidXanatos » Thu May 09, 2013 12:36 pm

- Can't choose Temp folder in 1st start wizard
-- this is intended behavior (not a bug), we need to force the incoming directory and the temp directory to be on the same physical drive.
- In 1st start wizard advanced options are not show, after turning off only grey out. It should be grey out and not invisible even first
-- removing the items again would be better i agree, but it would be more complicated to implement, and I believe it does not really matter if its grayed out or fully hidden. I will leave it as it is for the foreseeable future.
- Default columm size & order have no sense
-- fixed in 0.31 using settings you send me
- Lag switching windows that show previous window
-- fixed in 0.31 when switching and waiting for new file list window is cleared.
- torrent download find peer but don't download
-- fixed in 0.31 issue was only with *,torrents not with magnets
- No fileicon with magnet links after grabbed. It works after restart <--- tried more times and now no more happens, strange
-- fixed in 0.31
- AutoShare ed2k is active, but right click on single files it says not
-- fixed in 0.31, Note: after changing the setting it is not applyed to old files, also note that the state is being set only after the file was hashed and a ed2k has
- When removing/deleting a file from downloads, it is not canceled from Temp folder
-- fixed in 0.31
- When downloading multiple torrent files since the folder is not wrote on incoming folder there is a bug in filelist
-- fixed in 0.31
- single file sharing does not work
-- fixed in 0.31
