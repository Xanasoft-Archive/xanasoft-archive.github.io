I delete the file from "share",but it still exist in the Incoming.

I will look into that, iirc. you must remove the physical file manually, I added that as a safety measure when testing things.
I will change it so that you can physically delete complete files from incoming but still not from other shared directories.

I add ed2k & magnet & torrent (the same files) to NeoLoader,and NeoLoader's downloads displays the three same items.

Dose the magnet and the torrent head the same info hash?

That the ed2k hash couldn't be associated with the info hash is normal as long as no one have downloaded the file previously completely and waited for Neo to publish the association in the NeoKad.
And even if this happens NeoKad needs some time to find the other hashed, the the items ware inserted at once, this will cause them not to be properly associated.

I guess I should add a feature that when it find later on matching Hashes it offers the user to merge the 2 downloads.
