Note: This requires Sandboxie-Plus version 1.11.0 or later.

Sandboxie-Plus > Options > Global Options > [Support & Updates] > [Sandboxie Support]

There you paste the general certificate completely into the text box and apply it, by clicking apply, if you have not already done that. Once there is a valid certificate, you can enter the key for the Sandboxie-Plus Advanced Encryption Pack, on the same page. Below the text box, there is a link that opens a form to input an upgrade key.
