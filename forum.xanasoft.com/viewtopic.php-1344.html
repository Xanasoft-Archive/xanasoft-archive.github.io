Hi,

even in "Standard Isolation Sandbox"es when forcing a program into the sandbox (e.g. ForceProcess=EXCEL.EXE in ini section) all command line parameters are cut off.
e.g. calling excel mysheet.xlsx only opens only Excel with no sheet.

So how can I pass on command line parameters into the sandbox, like opening an excel sheet with Excel instead manually from within Excel?


Thank you

Re: How to pass on command line parameters into sandbox?

Posted: Wed Oct 25, 2023 6:18 pm
by bastik-1001
This could be done, by launching a sandboxed CMD in the same sandbox, or by using start.exe in Sandboxie's folder.

Code: Select all

start.exe /box:[yourboxnamehere] "[yourprocesshere].exe /[yourargumentshere]"
The square brackets are part of the placeholder.

Code: Select all

start.exe /box:DefaultBox "notepad.exe C:\test.txt"
