(Just more input as background information, this should not be helpful to you, besides understanding that it appears to be a technical issue.)

Sandboxie Plus and Classic, share the same core components, so whenever something works with Classic there is no reason why it should fail with Plus.

It is possible, that there is a regression in one of the versions that got released after the one that worked. Another possibility is that Windows 11 behaves different than Windows 10 and therefore any version of Sandboxie fails. Both cases are a bug.
