Sandboxie-Plus Roadmap

Posted: Sat Jan 20, 2024 8:42 pm
by bastik-1001
Over at the Wilders Secuirty Forums, David has posted a roadmap, some time ago.

This here is just a copy with items left to implement:
  • rework scm hooking to improve Windows 10 compatibility scheduled for 1.13.0
  • add missing virtualization to EventPair & KeyedEvent objects
  • add filtering and virtualization to Timer objects
  • add ability to force sandboxed processes to use a pre-defined socks 5 proxy available in insider build
  • add ability to intercept DNS queries such that they can be logged and/or redirected available in insider build
  • add support for EFS using the user proxy available in insider build
  • add break out document functionality available in insider build
  • Better UWP support, in compartment mode it's already possible to a limited extent to run store packaged win32 apps, but that is far from actual UWP/Store support.
  • Improved MSI installer support, to be run in a sandbox.
  • Rework service handling, implement a proper sandboxed service manager
  • Rework COM proxing EP mapper handling
  • Rework creation of the sandboxed token to improve compatibility
I will update this post, whenever something gets implemented or added to the list.
