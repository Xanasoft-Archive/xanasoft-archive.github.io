Unless the sandbox is set to be emptied and the extensions are not passed to the real system, they should remain present.

Sandboxie Plus 1.8.2 added a browser compatibility wizard, that allows you to improve support for browsers that are not fully recognize, It might be helpful in your case, too.

If it still does not work, with a recent version of the browser and Sandboxie, please share your configuration.
