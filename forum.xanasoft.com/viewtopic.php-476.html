Apologies - I just found the announcement of MajorPrivacy after typing my post.

A few thoughts that are related to development - I'm going to just think out loud for a moment.
  • MP needs to retain the granular control options with explanations of methodology used (GPO, registry, etc).
  • It might be useful to create setting "Playbooks" for various scenarios (so to avoid activating a setting that would stop an essential program from running). Some scenarios might include "Microsoft Office user, Non Microsoft Office user, VR Gamer PC, Video/Media Production PC, Virtualization User (host/guest), Programmers PC"
  • An option to run a "MajorPrivacy checkup" after microsoft forces an update, so that MP will detect any settings that broke or were reenabled by the update. Also, (would completely need to be an optional opt-in) it could be useful for users to send data on what sneaky settings microsoft adjusted on their PC by updates. Also, this would make good forum post content for the community. A link to the forum could be offered within MajorPrivacy.
And David, if you are going to beta test, let us know. Thank you for your work again.
