When it is possible to do that without Sandboxie, this should be possible while the software is sandboxed. By default, applications in sandboxes can see the processes of the host. The applications on the host are not prevented from accessing processes that run sandboxed, by default.

Processes running in two different sandboxes don't see each other, by default.

Re: Can you allow an application access a sandboxed application?

Posted: Fri Jan 26, 2024 4:47 pm
by bastik-1001
(This assumes that the graphical user interface Sandman, which is the default GUI for Sandboxie-Plus, is being used.)

You can disable that a sandboxed process can't see processes in other sandboxes in the Sandbox Options > Advanced Options > [Hide Processes] where you have to uncheck "Don't allow sandboxed processes to see processes running in other boxes". This is also the place where you can hide host processes from processes running in that sandbox.

Sandbox Options > Resource Access gives control over various resources that can be allowed or denied.
