This isn't the official continuation of the old Sandboxie and old software licenses are probably unavailable. It's also unclear whenever those could still be applied to the old software.

However, since the sources were released Sandboxie and Sandboxie Plus, got released without the previous restrictions. So running a recent version of Sandboxie or Sandboxie Plus allows you to use features for free, that users had to pay for, previously.

Sandboxie releases by David are free for non-commercial use. Some features that have been added to new releases require a supporter certificate, which can be purchased here or obtained by other means, like contributing to the project.
