I guess, David will be pleased to hear that.

Upon installing Sandboxie Plus, it will be detected that Sandboxie Classic is being installed and you are asked if you want to remove it. Maybe you would want to create a copy (aka backup) of your Sandboxie.ini, although it should still be present after you proceeded with the removal and installation process.

When you start up the new interface, you will see the sandboxes you have created before, being listed there. There are some options to change how it is going to look like. One of them is a vintage view, that tries to make it look like the older interface.
