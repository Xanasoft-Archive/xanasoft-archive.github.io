You can set it manually from the template file:
  1. Sandboxie Plus > Options > Edit Templates.ini
  2. Find: [Template_BlockSoftwareUpdaters]
  3. Under this section, add a # sign in front of the lines of the applications you want to update and save the file.
  4. Sandboxie Plus > Options > Reload configuration
