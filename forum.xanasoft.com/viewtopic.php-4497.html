It should be able to read the disc, but apparently it is not. If that is still an issue with newer software, you may want to try to open access to the drive.

Code: Select all

OpenFilePath=The path to the CD drive
or with

Code: Select all

OpenPipePath=again the path to the drive
if EAC is installed in the sandbox or the binary is located inside of it.
