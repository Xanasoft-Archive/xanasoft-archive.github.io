by bastik-1001
When you run your browser, with all its addons inside Sandboxie, they will have the same access as that instance of your browser. Only when you run an instance of Firefox (or any other browser) that runs without those addons within Sandboxie, either by not giving it access to the addon folder or by installing it into a dedicated box without access to the addons, you will be shielded from them.

Also note that anything that runs on the host or within a sandbox, can access what the user has access to, so that the system itself needs to be secure in the first place, although using Sandboxie for that purpose can still be beneficial.
