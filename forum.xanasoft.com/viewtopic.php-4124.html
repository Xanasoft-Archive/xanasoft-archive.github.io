Activation context generation failed for "C:\Sandbox\needleman\DefaultBox\drive\C\Program Files\Mozilla Firefox\firefox.exe". Dependent Assembly mozglue,language="&#x2a;",type="win32",version="1.0.0.0" could not be found. Please use sxstrace.exe for detailed diagnosis.

When I look on the web for advice on the problem, there are four suggestions:
--Reinstall application. Done
--Reinstall called application. Done
--Install all Windows C++ libraries. Done
--Check registry. The registry entry referenced is not in my registry.

Any suggestions deeply appreciated.
