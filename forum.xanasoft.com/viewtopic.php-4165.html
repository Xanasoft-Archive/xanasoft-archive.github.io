Posted: Thu Jun 16, 2022 7:29 pm

Windows update KB5013942 (May 2022) and KB5014699 (June 2022) prevent Sandboxie/Sandboxie+ from starting.
Result is error message SBIE2314 "canceling process start.exe"
Google search for Sandboxie issue related to KB5013942 is documented on the web.
The only solution I've found is to uninstall KB5013942 and KB5014699, then run MS program "wushowhide.diagcab" to prevent them from being automatically reinstalled.
* Does anyone have a better solution to this problem?
* Is the Sandboxie development team addressing this issue.
