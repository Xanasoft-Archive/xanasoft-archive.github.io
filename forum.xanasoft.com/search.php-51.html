You can set it manually from the template file: Sandboxie Plus > Options > Edit Templates.ini Find : [Template_BlockSoftwareUpdaters] Under this section, add a # sign in front of the lines of the applications you want to update and save the file. Sandboxie Plus > Options > Reload configuration
