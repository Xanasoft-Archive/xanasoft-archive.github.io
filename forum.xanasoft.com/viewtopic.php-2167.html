This is still relevant: SecureDeleteSandbox.

Sandboxie itself does not attempt to securely remove traces, as this can't be guaranteed and could give a false sense of privacy. You may want to read about Privacy Concerns as well.

It's better to use Sandboxie on an encrypted system or at least some encrypted partition. You could create a ram disk and create sandboxes on there, so everything gets deleted, once the ram disk is unmounted.
