It might not be worth your time to report on why it did not work for you, so I skip guessing or directly asking why that is the case for you.

Now to your problem.

It should work with the Windows internal procedure, but apparently that does not work for you.

Please try to uninstall it in a different way:
1) Although it should not be required, end the process Sandman.exe
2) Run "C:\Program Files\Sandboxie-Plus\unins000.exe" it should ask for admin rights which you have to grant
3) Select that you want to uninstall it
4) Choose if you want to keep the configuration AND/OR sandboxes (probably not, in your case.) WARNING: Files stored in sandboxes, will be removed with them.
