EDIT:
It doesn't seem possible to create and mount a new encrypted box directly from the command line. At least initially, it is necessary to mount the encrypted box and assign a password to it. After creating a password, you can mount encrypted boxes from the command line.

Example:

Code: Select all

"C:\Program Files\Sandboxie-Plus\Start.exe" /box:BoxName /key:BoxPassword /mount_protected

"C:\Program Files\Sandboxie-Plus\Start.exe" /box:BoxName "program.exe"

"C:\Program Files\Sandboxie-Plus\Start.exe" /box:BoxName /unmount
