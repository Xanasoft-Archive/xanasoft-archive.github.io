Your issue is that your new installation is UEFI, but the iso boot loader is BIOS/MBR only, if you did nto install an UEFI bootloader to the HDD or created a bootloader UEFI usb stick you will not be able to boot.

You can however create a appropriate UEFI usb stick on any PC and use it with any other pc.
