Re: Can the portable version overwrite itself?

Posted: Sat Jul 15, 2023 5:00 pm
by bastik-1001
The installer allows you to extract the files to a folder you chose. This can be a different folder. If you choose to extract the files to an existing version of portable Sandboxie-Plus, the installer will prompt if you want to use that folder. After approving to use that folder, files will be overwritten. Otherwise, you can choose a different folder.

Of course, only files that are not used can be replaced.
