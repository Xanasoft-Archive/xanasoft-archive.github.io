I can't test if this works as intended, but you should be able to use different compatibility modes for the same executable.
  1. Open a sandboxed explorer window of the sandbox you want to start the app with different compatibility settings.
  2. Navigate to the file you intend to run with compatibly settings, right click on it and select to open its properties.
  3. Go to the compatibly tab and make the changes and apply them.
  4. Launch the program sandboxed.
Now, the properties outside the sandbox should remain the same. When you have more than one sandbox, each sandbox can have a different compatibility mode for a program. I could test if I can make those changes, but I can't test if compatibility mode is indeed applied.

If it does not work, it might be that Sandboxie is not, or not yet compatible with such functionality, even though that isn't new.
