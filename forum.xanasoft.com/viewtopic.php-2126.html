Yes, you can open files from external drives in Sandboxie. You might open just the files or start an instance of the explorer with Sandboxie from which you can access the files.

It is still problematic to attach devices you deem untrustworthy (or that are of unknown origin) since it might not be a drive or the file handler for the preview might be tricked into executing something.
