Here is an overview of the resources access types.

Write File Path is the type of rule that has to be used in order to have a sandboxed program being able to write to a sandboxed location, but not being able to read from the unsandboxed source. This makes the files on the host invisible to sandboxed processes, unless they are exempt by the user's choice.
