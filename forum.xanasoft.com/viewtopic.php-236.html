Sandboxie-Plus v1.12.7

Usually I use Firefox in Sandboxie, thought that I would try Chrome. Chrome works just fine except for one thing. Every time that I open up Chrome inside Sandboxie, I receive a message that says "Chrome didn't shut down correctly." It also asks me if I want to "Restore pages".

When I open Chrome regular, with no Sandboxie, I do not receive that message, only when using Sandboxie. Then I tried Brave browser and I am receiving the exact same messages. Any idea what's happening?

This is a known issue, for which there is no solution, yet, nor do I know a workaround.

Re: Message when using Chrome and Brave

Posted: Mon Jan 29, 2024 9:26 pm
by 4techniques
Problem solved. It was the Norton Safe Web extension that I had add to both Chrome and Brave. I removed that extension from both browsers, replaced it with AVG Online Security, everything is now working just fine.
Thanks,
4techniques
