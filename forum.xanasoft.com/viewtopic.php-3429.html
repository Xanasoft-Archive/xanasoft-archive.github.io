Posted: Thu Nov 17, 2022 5:45 pm
by bogdan.lazea
Problem is fixed with version 1.5.3.
When I installed Windows 11 OS fresh copy, install sandboxie non portable, but in settings put the location of sandboxes into C:\Users\<user>\Sandboxes folder. Then after deleting Default sandbox restart your PC.
After that I installed my Bitdefender Security Suite.

Now Windows 11 and Bitdefender work as expected, and I am able to create Hardenened Security sandboxes and open Firefox inside.

This ticket can be closed,
Cheers
