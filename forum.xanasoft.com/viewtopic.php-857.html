Is there any way to move a Sandbox folder to other disk? All the cloned content is weighing too much on the disk it's installed, but I can't reinstall and create all the Sandboxes again, so I wanna just move all the content.

Yes, you can do that. You have to set a custom file root path for the box or boxes (per box), then copy the files to the location.

Documentation about the file root path.

You have to edit the Sandboxie.ini (usually located in C:\Windows) in the section of the sandbox(es) you want to move.

Code: Select all

FileRootPath=X:\Sandbox\your_sandbox_name
where X is the drive letter and "your_sandbox_name" is the name of your sandbox, although it can be labeled differently. Sandboxie itself does not move the data over, you have to move them manually or with your favorite tool.

When you use Sandboxie-Plus you can edit the relevant section of the .ini from within the user interface. Right-click on the sandbox in the GUI of Sandboxie-Plus (Sandman), select "Sandbox Options" and then "Edit ini Section" which should be at the bottom of the entries.

The GUI (Sandman) allows setting a global file root path, which by default is set to

Code: Select all

\??\%SystemDrive%\Sandbox\%USER%\%SANDBOX%
but it can be changed under Options > Global Settings > Advanced Config > Sandboxie Config

It can also be changed manually by editing the Sandboxie.ini
