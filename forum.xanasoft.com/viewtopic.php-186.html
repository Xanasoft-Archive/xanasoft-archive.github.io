Part of the documentation is a guide for users of the previous Sandboxie and Sandboxie Classic for the migration to the new graphical user interface (GUI) of Sandboxie Plus.

This helps to locate options and features in the new user interface, that were present in the old user interface, as the guide provides pictures on where one has clicked before and where this feature can be found. Users that start fresh with Sandboxie Plus, might find the overview helpful as well, although the general user interface should be easy to understand.
