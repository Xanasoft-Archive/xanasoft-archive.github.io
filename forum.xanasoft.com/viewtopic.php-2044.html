Is the sandboxed application installed on the system (or located on the system, instead of being only in the sandbox)? If so, setting a resource to open should enable the resource to be accessed.

However, if the application is installed to the sandbox or downloaded to it, then "Open for All" has to be used to be applicable for programs located inside the box, as well as those outside.
