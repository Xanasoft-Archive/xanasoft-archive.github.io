Right now, with Sandman (the default user interface for Sandboxie Plus), you can create shortcuts per sandbox for programs, like your browser, but also to the Windows explorer (explorer.exe).

So you can create a shortcut to explorer.exe for "DefaultBox", "Yourcustomboxname1", "Box2", etc. to be placed on your Desktop (or another folder of your choice) to execute explorer.exe in the box to which the shortcut points to.
