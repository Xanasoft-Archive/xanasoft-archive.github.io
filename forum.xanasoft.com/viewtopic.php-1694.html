Hardware acceleration doesn't work in Firefox (and other browsers) when running in Sandboxie.

Posted: Fri Apr 28, 2023 10:33 pm
by W1K23
For a long time, I was angry with Firefox for playing YouTube videos with a much higher CPU usage than Edge, Chrome. I thought Youtube was optimized for chromium. Today I finally found the cause of the problem. I always run Firefox in Sandboxie. Hardware acceleration doesn't work in SBoxie. Therefore the CPU usage is 3 times higher and my computer gets very hot while playing 4K videos. When Firefox is started in the normal way, there is no high CPU usage and the notebook doesn't heat up, the fan doesn't spin.
CPU usage is same high under Edge and Chrome in Sandboxie while playing YouTube videos.
How to turn on Firefox hardware acceleration in Sandboxie?

Firefox by default
about:support
Graphics - Compositing: WebRender

Firefox in Sandboxie
Graphics - Compositing: WebRender (software)
