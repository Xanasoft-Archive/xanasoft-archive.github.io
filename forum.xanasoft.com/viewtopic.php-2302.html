Yes, you can use CCleaner (or other tools) to wipe out the files created within those sandbox folders. Preferably, when those sandboxes aren't active.

Please note, that this is not required to defend against common malware, since the normal way of deletion does not allow the file to be executed by the OS or some application. It may defend against data-leakage, since files deleted by the OS can be recovered.

It can't be ensured, that overwriting files, gets rid of all information, like when something is being copied or cached. Though, overwriting files still makes it harder to recover information. One downside, is that the more data being written, the more it shortens the lifespan of the drive and takes longer to being completed, of course.
