Does this issue still occur with newer versions?

Unless the sandbox is configured to be autodeleted, when the last process is terminated, the files and folders should stay present.

Those messages are to be expected when Firefox is getting updated, which should not be done while being sandboxed, so they just tell you to update the browser outside Sandboxie's control and run it again under Sandboxie's control. It advises you to clear the sandbox data, since the download of the update is still included in the sandbox and could trigger messages again.
